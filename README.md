# Jonahsnider Tap

## How do I install these formulae?

`brew install jonahsnider/tap/<formula>`

Or `brew tap jonahsnider/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jonahsnider/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
