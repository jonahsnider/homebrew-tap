# Homebrew Tap

A [Homebrew](https://brew.sh/) tap for my tools.

## Supported tools

- [how](https://github.com/jonahsnider/how) - Learn how to use CLI apps
  - `brew install jonahsnider/tap/how`

## How do I install these formulae?

`brew install jonahsnider/tap/<formula>`

Or `brew tap jonahsnider/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jonahsnider/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
